Subscription Fintech — FP&A Model & KPI Dashboard

Tools: Excel, Power BI, SQL | Scope: 24‑month model & CFO dashboard | Data: synthetic

Executive summary

Built a driver-based model for a subscription fintech (new subs, churn, ARPU, CAC, support cost). Published a Power BI dashboard with variance waterfall, cohorts, and cash runway. Key finding: a 1pp churn improvement extends runway by 5 months with current CAC and pricing; recommended action is a retention program targeting high-cost cohorts.

Problem & context

  • Goal: produce a 24‑month P&L, cash plan, and KPI pack
  • Drivers: pricing, discounts, churn, CAC, FX, headcount
  • Constraints: limited historicals → scenario ranges

Data & method

  • Excel model with dynamic arrays; separate inputs/logic/outputs
  • SQL window functions to build monthly retention cohorts
  • Power BI model (star schema) and DAX time intelligence

Results

  • Variance waterfall Plan→Actual→Forecast
  • KPIs: MRR/ARR, churn, CAC payback, margin
  • Sensitivity: churn ±2pp, CAC ±10%, FX ±5%

What I'd do next

Add product‑level pricing tests and retention drivers; build cohort LTV with marketing channel granularity.
